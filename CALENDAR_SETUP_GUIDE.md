# カレンダー機能セットアップガイド

## 概要
お客様がホットペッパービューティーの予約状況をリアルタイムで確認できるカレンダー機能を追加しました。

## 実装内容

### 1. パッケージのインストール
以下のパッケージが追加されました：
```bash
npm install react-calendar google-auth-library googleapis react-big-calendar moment
```

### 2. 追加されたファイル

#### サービス層
- `src/services/calendarService.js` - Google Calendar API連携サービス
  - Google Calendar APIの初期化
  - イベント取得・フォーマット
  - ホットペッパービューティー予約の識別
  - 予約詳細情報の解析
  - 利用可能時間の計算
  - 月間統計の生成

#### コンポーネント層
- `src/components/Calendar.js` - カレンダー表示コンポーネント
- `src/components/Calendar.css` - カレンダーのスタイル

#### ページ層
- `src/pages/CalendarPage.js` - カレンダーページ
- `src/pages/CalendarPage.css` - ページのスタイル

### 3. 修正されたファイル
- `src/App.js` - カレンダーページのルーティング追加
- `src/components/Header.js` - ナビゲーションに「予約状況」ボタン追加
- `env.example` - Google Calendar API設定を追加

## 機能詳細

### カレンダー表示機能
- 月表示、週表示、日表示の切り替え
- ホットペッパービューティー予約と直接予約の色分け表示
- 予約詳細のモーダル表示
- 日本語ローカライゼーション

### 予約状況表示
- 選択日の利用可能時間表示
- 予約済み時間の表示
- 月間予約統計（総予約数、予約方法別、サービス別）

### 予約方法案内
- ホットペッパービューティーへのリンク
- 電話予約
- LINE予約

## セットアップ手順

### 1. Google Calendar API設定

#### Google Cloud Console設定
1. [Google Cloud Console](https://console.cloud.google.com/)にアクセス
2. 新しいプロジェクトを作成または既存プロジェクトを選択
3. Google Calendar APIを有効化
4. 認証情報 > APIキーを作成
5. APIキーの制限を設定（HTTPリファラー）

#### Calendar ID取得
1. Google Calendarで対象のカレンダーを選択
2. 設定と共有 > カレンダーIDをコピー

### 2. 環境変数設定
`.env`ファイルに以下を追加：
```env
REACT_APP_GOOGLE_API_KEY=your_google_api_key_here
REACT_APP_GOOGLE_CALENDAR_ID=your_calendar_id_here
```

### 3. ホットペッパービューティー連携設定
1. ホットペッパービューティーの管理画面にログイン
2. 「外部カレンダー連携」または「Google Calendarとの連携」機能を有効化
3. 手順に従ってGoogle Calendarと連携

## 予約情報の解析

### ホットペッパービューティー予約の識別
以下のキーワードでホットペッパービューティーからの予約を自動識別：
- 「ホットペッパー」
- 「hotpepper」
- 「hot pepper」
- 「ホットペッパービューティー」

### 予約詳細の抽出
イベントの説明欄から以下の情報を自動抽出：
- 顧客名
- サービス内容
- 電話番号
- 備考

## トラブルシューティング

### よくある問題

#### 1. カレンダーが表示されない
- Google Calendar APIキーが正しく設定されているか確認
- Calendar IDが正しいか確認
- カレンダーが公開設定になっているか確認

#### 2. 予約情報が表示されない
- カレンダーにイベントが正しく追加されているか確認
- ホットペッパービューティーとGoogle Calendarの連携が正常か確認

#### 3. API制限エラー
- Google Cloud ConsoleでAPI使用量を確認
- 必要に応じてクォータの増加を申請

### デバッグ方法
ブラウザの開発者ツールコンソールで以下の情報を確認：
- Calendar Service初期化状況
- API呼び出しのエラー詳細
- 取得したイベントデータ
