# 【店舗用LINE公式アカウント】予約運用マニュアル（仮予約制）

## 1. 運用の方針

他の予約サイト（ホットペッパー等）とのダブルブッキング（重複予約）を防ぐため、LINEでは即時予約確定を行いません。
**「空き状況の確認・仮申し込み」**として受け付け、**店舗側が空きを確認して返信した時点で「予約確定」**となるフローを採用します。

### 予約確定までの流れ

1. **お客様**：リッチメニューの「予約」ボタンを押す
2. **お客様**：表示されたテンプレート（希望日時など）を入力して送信
3. **LINE（自動）**：「確認しますのでお待ちください」と自動返信
4. **店舗（手動）**：施術の合間に他の予約表を確認
5. **店舗（手動）**：「〇月〇日〇時でご予約承りました」と返信 → **ここで確定**

---

## 2. LINE設定内容（コピー用）

以下の3つの設定を行うことで、この運用が可能になります。

### ① リッチメニュー（ボタン）の設定

トーク画面下部のメニューに「空き状況確認・予約」ボタンを配置し、タップ時のアクションに以下のテキストを設定します。

**【ボタンのアクション：テキスト】**

```
【空き状況の確認・予約お申込み】
※現時点では予約確定ではありません。

下記をご入力の上、送信してください。
（コピーや引用をして入力の手間を省けます）

■お名前（フルネーム）：

■ご来店回数：
（ 初めて ／ 2回目以降 ）

■ご希望コース（または分数）：
（例：60分ベーシック、耳つぼなど）

■ご希望日時（第3希望まで頂けるとスムーズです）：
1. 〇月〇日 〇〇時〜
2. 
3. 

■その他（耳の不調・悩み、通院歴など）：
```

### ② 自動応答メッセージの設定

お客様から予約の連絡が来た瞬間に、自動で返すメッセージです。

- **応答タイプ**：キーワード応答（部分一致）
- **キーワード**：「予約」「申込み」「空き状況」「希望日時」

**【自動返信メッセージ文面】**

```
ご連絡ありがとうございます！
〇〇（店名）です。

メッセージを受け付けました。

⚠️まだご予約は確定しておりません⚠️

現在、担当者が他の予約サイト等の空き状況と照らし合わせて確認しております。

お席を確保でき次第、改めてこちらから「ご予約確定」のメッセージをお送りしますので、いましばらくお待ちくださいませ。

※施術中はお返事が遅くなる場合がございます。
※お急ぎの方はお電話にてご連絡ください。
📞 03-xxxx-xxxx
```

### ③ あいさつメッセージの設定

友だち追加された瞬間に送るメッセージです。

**【あいさつメッセージ文面】**

```
友だち追加ありがとうございます！
イヤーエステ・耳そうじ専門店の〇〇です👂✨

ご予約や空き状況の確認は、
下のリッチメニュー（ボタン）から簡単に行えます。

施術中はお電話に出られないことが多いため、こちらのLINEからメッセージをいただけると助かります！

⚠️ご注意
耳の炎症・傷・感染症の疑いがある場合や、飲酒をされている場合は施術をお断りすることがございます。予めご了承ください。
```

## 3. 運用時の注意点（オーナー様へ）

### 通知の確認

お客様からメッセージが届くと、管理アプリ（LINE Official Account Manager）に通知が来ます。「未読」になっているメッセージは必ず確認してください。

### 手動返信のタイミング

施術が終わったら、まず予約表（他サイト含む）を確認し、LINEのお客様へ返信をお願いします。

- **空いていた場合**：「ご予約ありがとうございます。〇月〇日〇時で承りました。ご来店お待ちしております！」
- **埋まっていた場合**：「申し訳ございません。その時間は埋まっております。〇時〜なら空いておりますがいかがでしょうか？」

### 「完了」ステータスへの変更

予約が確定し、返信が終わったチャットは、管理画面で「対応済み（完了）」にしておくと、誰に対応していないかが一目で分かります。
