## Google Adsとは

Google Ads（旧Google AdWords）は、Google検索結果やYouTube、Google検索パートナーサイトなどに広告を表示して、自分のウェブサイトへのアクセスを増やすことができる広告配信サービスです。

### 広告の種類

1. **検索広告**: Google検索結果に表示されるテキスト広告
2. **ディスプレイ広告**: ウェブサイトやアプリに表示される画像・動画広告
3. **YouTube広告**: YouTube動画の前後に表示される動画広告
4. **ショッピング広告**: 商品画像と価格が表示される広告
5. **ローカル広告**: 地図検索結果に表示される店舗情報広告

### 料金体系

- **CPC（Cost Per Click）**: 広告がクリックされたときに課金
- **CPM（Cost Per Mille）**: 広告が1000回表示されたときに課金
- **CPA（Cost Per Acquisition）**: コンバージョン（予約・問い合わせ）が発生したときに課金

### 予算設定

- **1日の予算**: 1日あたりの最大支出額を設定可能
- **入札戦略**: 自動入札または手動入札を選択可能
- **最低入札額**: キーワードごとに最低入札額が設定される

## Google Adsアカウント作成

### ステップ1: Google Adsアカウント作成

1. [Google Ads](https://ads.google.com/)にアクセス
2. 「今すぐ始める」をクリック
3. Googleアカウントでログイン
4. ビジネス情報を入力：
   - ビジネス名: yoon²ゆんゆん
   - ウェブサイトURL: `https://yoon2.com`
   - 国・地域: 日本
   - タイムゾーン: 日本標準時（JST）
5. 支払い情報を入力（クレジットカードまたは銀行口座）

### ステップ2: 広告アカウントIDの取得

1. Google Adsダッシュボードにログイン
2. 右上の設定アイコン（⚙️）をクリック
3. 「アカウント設定」を選択
4. **顧客ID（10桁の数字）**を確認
   - 例: `123-456-7890`
   - このIDを環境変数に設定します

## コンバージョン追跡の設定

### ステップ1: コンバージョンアクションの作成

1. Google Adsダッシュボードにログイン
2. 「ツールと設定」→「測定」→「コンバージョン」
3. 「+」ボタンをクリック
4. コンバージョンアクションを選択：
   - **ウェブサイト**: 予約フォーム送信、問い合わせフォーム送信など
5. コンバージョン名を入力：
   - 例: 「予約フォーム送信」
6. **コンバージョンID**と**ラベル**をコピー

### ステップ2: コンバージョンIDの設定

`.env`ファイルに以下を追加：

```env
# Google Ads コンバージョン設定
REACT_APP_GOOGLE_ADS_CONVERSION_ID=AW-1234567890
REACT_APP_GOOGLE_ADS_CONVERSION_LABEL=AbC-dEfG-hIjK-lMnO
```

### ステップ3: コンバージョン追跡の実装

予約フォーム送信時にコンバージョンを記録：

```javascript
import { trackConversion } from '../services/googleAdsService';

// 予約フォーム送信時
const handleSubmit = async (formData) => {
  // フォーム送信処理
  await submitBooking(formData);
  
  // コンバージョンを記録
  trackConversion('booking_submit', {
    value: 1.0,
    currency: 'JPY'
  });
};
```

### ステップ4: コンバージョンの確認

1. Google Adsダッシュボードにログイン
2. 「ツールと設定」→「測定」→「コンバージョン」
3. コンバージョンアクションを選択
4. 「テスト」タブでテストコンバージョンを送信
5. 24時間以内にコンバージョンが記録されることを確認

## 広告運用のベストプラクティス

### 1. キーワード選定

**推奨アプローチ**:
- **ロングテールキーワード**: 「イヤーエステ 松山 当日予約」など具体的なキーワード
- **地域キーワード**: 「松山」「愛媛」「北久米」など地域を含める
- **競合キーワード**: 競合店の名前を含むキーワードは避ける

### 2. 広告文の最適化

**効果的な広告文の要素**:
- ✅ 店舗名（yoon²ゆんゆん）
- ✅ 主要サービス（イヤーエステ、耳つぼ）
- ✅ 地域情報（松山、北久米）
- ✅ 特典・メリット（当日予約OK、駐車場完備）
- ✅ 明確な行動喚起（予約する、問い合わせる）

### 3. ランディングページの最適化

**推奨事項**:
- 広告の内容と一致した情報を表示
- 予約ボタンを目立つ位置に配置
- モバイル対応を確認
- ページ読み込み速度を最適化（3秒以内）

### 4. 予算配分

**推奨予算配分**:
- **検索広告**: 70%（最も効果的）
- **ディスプレイ広告**: 20%
- **YouTube広告**: 10%（動画コンテンツがある場合）

### 5. 入札戦略

**推奨設定**:
- **初期**: 「コンバージョン数の最大化」（自動入札）
- **データ蓄積後**: 「コンバージョン単価の目標値」に変更

### 6. 広告のA/Bテスト

**テスト項目**:
- 広告文のバリエーション
- ランディングページ
- 入札価格
- 広告表示時間帯

### 7. パフォーマンス監視

**監視すべき指標**:
- **インプレッション数**: 広告が表示された回数
- **クリック数（CTR）**: クリック率（目標: 2%以上）
- **コンバージョン数**: 予約・問い合わせ数
- **コンバージョン単価（CPA）**: 1件のコンバージョン獲得にかかる費用
- **ROAS（投資対効果）**: 広告費用に対する売上の比率

## トラブルシューティング

### 広告が表示されない

**原因と対処法**:

1. **審査未通過**
   - Google Adsダッシュボードで審査状況を確認
   - 審査通過まで待つ（通常24時間以内）

2. **予算不足**
   - 1日の予算を確認
   - 入札価格が高すぎる可能性がある

3. **キーワードの品質スコアが低い**
   - 広告文とランディングページの関連性を確認
   - キーワードをより具体的に変更

### コンバージョンが記録されない

**原因と対処法**:

1. **トラッキングコードが正しく実装されていない**
   - `index.html`のGoogle Adsタグを確認
   - ブラウザの開発者ツールでネットワークリクエストを確認

2. **コンバージョンIDが間違っている**
   - `.env`ファイルの`REACT_APP_GOOGLE_ADS_CONVERSION_ID`を確認
   - Google Adsダッシュボードで正しいIDを確認

3. **コンバージョンラベルが設定されていない**
   - `.env`ファイルの`REACT_APP_GOOGLE_ADS_CONVERSION_LABEL`を確認
   - コンバージョンアクション設定でラベルを確認

### クリック単価（CPC）が高い

**改善方法**:

1. **キーワードの見直し**
   - 競合の多いキーワードを避ける
   - より具体的なロングテールキーワードを使用

2. **広告文の最適化**
   - より関連性の高い広告文を作成
   - 品質スコアを向上させる

3. **ランディングページの最適化**
   - ページ読み込み速度を改善
   - コンテンツの関連性を向上

### コンバージョン単価（CPA）が高い

**改善方法**:

1. **ランディングページの最適化**
   - 予約フォームを目立つ位置に配置
   - ページの読み込み速度を改善

2. **広告文の見直し**
   - より具体的な情報を提供
   - ユーザーの意図に合った広告文を作成

3. **ターゲット設定の見直し**
   - 地域設定を最適化
   - 時間帯設定を調整
