# 画像仕様書

## 概要
各画面に追加する画像の推奨サイズ、解像度、内容についての詳細仕様です。

---

## 1. ホーム画面

### 1.1 タイトルのバックグラウンド画像（Hero Section）

**配置場所**: ホーム画面の最上部、タイトル「yoon²」の背景

**表示サイズ**:
- デスクトップ: ほぼ全画面（`min-height: calc(100vh - 100px)`）
- タブレット: ほぼ全画面
- モバイル: ほぼ全画面

**推奨画像サイズ**:
- **幅**: 1920px（フルHD）
- **高さ**: 1080px（フルHD）
- **アスペクト比**: 16:9
- **解像度**: 72-96 DPI（Web用）

**ファイル形式**: 
- JPEG（推奨）または PNG
- ファイルサイズ: 500KB以下を推奨（読み込み速度のため）

**画像内容の推奨**:
- **テーマ**: リラクゼーション、上質な空間
- **構図**: 
  - 中央にテキストが配置されるため、中央部分は比較的シンプルに
  - 左右に装飾的な要素を配置
- **色調**: 
  - 現在のサイトカラー（ベージュ系: `#F4F1EB`）と調和する色
  - 明るめのトーン（テキストが読みやすくなるように）
- **具体的なイメージ**:
  - サロンの外観やエントランス（柔らかい光）
  - リラクゼーション空間の雰囲気（間接照明、キャンドルなど）
  - 抽象的なパターンやテクスチャ（現在のSVGパターンに近い）
  - イヤーエステや耳つぼに関連するアイテムを美しく配置

**注意事項**:
- テキストが重なるため、中央部分は明るく、コントラストが低め
- オーバーレイ（`rgba(244, 241, 235, 0.1)`）が適用されるため、少し濃いめでも問題なし
- レスポンシブ対応のため、上下左右に余白を持たせる

### 1.2 店内の様子（About Section）

**配置場所**: ホーム画面の「サービスの特徴」セクション内、右側

**表示サイズ**:
- デスクトップ: `height: 400px`、幅はグリッドの1/2
- タブレット: `height: 400px`、幅はグリッドの1/2
- モバイル: `height: 300px`、幅は全幅

**推奨画像サイズ**:
- **幅**: 1200px
- **高さ**: 800px
- **アスペクト比**: 3:2
- **解像度**: 72-96 DPI（Web用）

**ファイル形式**: 
- JPEG（推奨）または PNG
- ファイルサイズ: 300KB以下を推奨

**画像内容の推奨**:
- **テーマ**: サロンの実際の店内の様子
- **構図**: 
  - 横長の構図
  - 施術スペース全体が写るように
  - 左右どちらかに少し余白を持たせる（テキストとのバランス）
- **色調**: 
  - 明るく、清潔感のある印象
  - サイトカラーと調和する色
- **具体的なイメージ**:
  - 施術チェアやベッドが配置された空間
  - リラックスできる雰囲気のインテリア
  - 間接照明が効いた落ち着いた空間
  - 清潔感と上質感が伝わる写真

**注意事項**:
- 人物が写っている場合は、プライバシーに配慮（後ろ姿、シルエットなど）
- 商品やサービスが分かりやすく写っていると良い
- 角丸（`border-radius: var(--radius-lg)`）で表示される

### 3.2 サロンの様子（コースカード内）

**配置場所**: 各コースカード内の画像

**表示サイズ**:
- デスクトップ: `height: 522px`、幅はカードの全幅
- タブレット: `height: 522px`、幅はカードの全幅
- モバイル: `height: 522px`、幅はカードの全幅

**推奨画像サイズ**:
- **幅**: 800px
- **高さ**: 1000px
- **アスペクト比**: 4:5（縦長）
- **解像度**: 72-96 DPI（Web用）

**ファイル形式**: 
- JPEG（推奨）または PNG
- ファイルサイズ: 200KB以下を推奨（複数枚表示されるため）

**画像内容の推奨**:
- **テーマ**: 各コースに関連する具体的なイメージ
- **構図**: 
  - 縦長の構図
  - 中央にメインの被写体を配置
  - 上下に余白を持たせる
- **色調**: 
  - 各コースの特徴に合わせた色調
  - 統一感を持たせつつ、個性も出す
- **具体的なイメージ**:
  - **おすすめメニュー**: 高級感のある施術の様子、特別感のある空間
  - **耳つぼメニュー**: 耳つぼの施術の様子、耳つぼジュエリー
  - **イヤーエステ**: イヤーエステの施術の様子、道具
  - **ドライヘッドスパ**: ヘッドスパの施術の様子、リラックスしている様子

**注意事項**:
- 各コースごとに異なる画像を用意することを推奨
- 人物が写っている場合は、プライバシーに配慮
- `object-fit: cover`で表示されるため、上下が少しトリミングされる可能性がある

## まとめ

### 画像の配置場所とファイル名の推奨

```
public/
  images/
    hero/
      home-hero-background.jpg      # ホーム画面のHero背景
      shop-hero-background.jpg       # 店舗情報画面のHero背景（ホームと同じでも可）
      course-hero-background.jpg     # コース情報画面のHero背景（ホームと同じでも可）
    shop/
      shop-interior.jpg              # ホーム画面の「店内の様子」
      salon-overview.jpg             # 店舗情報画面の「サロンの様子」
    courses/
      course-recommend.jpg           # おすすめメニューのコース画像
      course-mimitubo.jpg            # 耳つぼメニューのコース画像
      course-ear-este.jpg            # イヤーエステメニューのコース画像
      course-dry-head.jpg            # ドライヘッドスパメニューのコース画像
```

### 画像最適化の推奨

1. **圧縮**: TinyPNG、ImageOptim、Squooshなどのツールで最適化
2. **WebP形式**: 対応ブラウザが多いため、WebP形式も用意すると良い
3. **遅延読み込み**: 既に`loading="lazy"`が実装されているため、自動的に適用される

### カラーパレットの参考

現在のサイトカラー:
- プライマリ: `#F4F1EB`（ベージュ）
- アクセント: `#C9A96E`（ゴールド）
- テキスト: `#2C2C2C`（ダークグレー）

画像はこれらの色と調和する色調を推奨します。
