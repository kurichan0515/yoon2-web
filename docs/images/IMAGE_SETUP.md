# 画像ファイルの配置手順

## 概要
各画面に画像を配置するための手順です。

## 必要な画像ファイル

以下の画像ファイルを `public/images/` ディレクトリに配置してください：

### Hero背景画像（1920px × 1080px推奨）
- `public/images/hero/head-spa.png` - ホーム画面と店舗情報画面のHero背景（ヘッドマッサージの施術風景）

### 店内・サロン画像（1200px × 800px推奨）
- `public/images/shop/play-room.png` - ホーム画面の「店内の様子」と店舗情報画面の「サロンの様子」（ミニマルな施術ルーム - ベッドと大きな窓）

### コース情報画面のHero背景画像（1920px × 1080px推奨）
- `public/images/hero/wait-room.png` - コース情報画面のHero背景（リラックスしたコーナー - アームチェアとサイドテーブル）

## 画像ファイルの配置

1. 提供された3つの画像ファイルを以下のように配置してください：

   **head-spa.png（ヘッドマッサージの施術風景）**:
   - `public/images/hero/head-spa.png` として保存
   - ホーム画面と店舗情報画面のHero背景に使用

   **play-room.png（ミニマルな施術ルーム）**:
   - `public/images/shop/play-room.png` として保存
   - ホーム画面の「店内の様子」と店舗情報画面の「サロンの様子」に使用

   **wait-room.png（リラックスしたコーナー）**:
   - `public/images/hero/wait-room.png` として保存
   - コース情報画面のHero背景に使用

2. 画像ファイルの最適化（推奨）:
   - TinyPNG、ImageOptim、Squooshなどのツールで最適化
   - ファイルサイズを小さくして読み込み速度を向上

## 実装済みの機能

以下の機能が実装済みです：

1. **ホーム画面**:
   - Hero背景画像の表示（head-spa.png）
   - 「店内の様子」画像の表示（play-room.png）
   - 画像読み込みエラー時のフォールバック表示

2. **店舗情報画面**:
   - Hero背景画像の表示（head-spa.png）
   - 「サロンの様子」画像の表示（play-room.png）
   - 画像読み込みエラー時のフォールバック表示

3. **コース情報画面**:
   - Hero背景画像の表示（wait-room.png）
   - レスポンシブデザイン対応

## 画像が表示されない場合

画像が表示されない場合は、以下を確認してください：

1. ファイルパスが正しいか確認
2. ファイル名が正確か確認（大文字小文字を含む）
3. ファイル拡張子が`.png`か確認
4. ブラウザのコンソールでエラーを確認
5. 画像ファイルが実際に存在するか確認

画像が読み込めない場合は、プレースホルダーが自動的に表示されます。

## ファイル構造

```
public/
  images/
    hero/
      head-spa.png      # ヘッドマッサージの施術風景
      wait-room.png      # リラックスしたコーナー
    shop/
      play-room.png      # ミニマルな施術ルーム
```
