# デザイン案生成プロンプト（簡潔版）

このプロンプトを画像生成AIやデザイン提案AIに入力して、yoon²イヤーエステサロンの新しいデザイン案を生成してください。

---

## プロンプト本文

yoon²（ユンユン）イヤーエステサロンのホームページの新しいデザイン案を作成してください。

### サイト情報
- **業種**: イヤーエステ・耳つぼ・ドライヘッドスパサロン
- **対象**: 20代〜40代女性中心
- **主要ページ**: ホーム、店舗情報、予約状況、予約・お問い合わせ、管理者ダッシュボード

### 現在のデザイン（変更前）
- **テーマ**: ナチュラル・エレガント
- **カラー**: ベージュ（#F4F1EB）、ゴールド（#C9A96E）、白
- **特徴**: 暖色系、低彩度、柔らかい印象、ソフト・マットなトーン

### 新しいデザイン要件
- **テーマ**: モダン・クール・エレガント
- **カラー**: **黒・青・白**を基調
- **特徴**: 
  - 色温度低め（冷色系、落ち着いた印象）
  - 落ち着いてかっこいい
  - モダンで洗練された
  - ミニマルデザイン
  - ダークモードを基調
  - タイポグラフィ重視
  - 余白を活かした構成

### デザイン案に含める要素

1. **カラーパレット**
   - 黒（背景、メイン）
   - 青（セカンダリ、アクセント）※具体的な色相を提案
   - 白（テキスト、アクセント）
   - HEXコード、RGB値、使用用途

2. **タイポグラフィ**
   - 日本語・英語対応のフォント提案
   - フォントサイズ階層
   - 見出し・本文のスタイル

3. **レイアウト**
   - Heroセクション（黒背景、青のアクセント）
   - セクション間の余白と配置
   - カードコンポーネント（ダークテーマ）
   - ボタンデザイン（青のアクセント）

4. **UIコンポーネント**
   - ヘッダー・ナビゲーション（黒背景）
   - ボタン（通常・ホバー状態）
   - フォーム要素（ダークテーマ）
   - カード・モーダル

5. **インタラクティブ要素**
   - ホバーエフェクト（青のグロー効果など）
   - トランジション・アニメーション
   - スクロールエフェクト

6. **レスポンシブ**
   - モバイル・タブレット・デスクトップ対応

### 出力形式
1. デザインコンセプト説明
2. カラーパレット（HEXコード付き）
3. タイポグラフィ
4. 各ページのレイアウト説明
5. UIコンポーネントデザイン
6. インタラクティブ要素
7. レスポンシブ対応

### 制約
- 既存コンテンツは変更しない
- ページ構造は維持
- アクセシビリティ（WCAG 2.1 AA）を満たす
- モバイルファースト

### 要望
- 上品で信頼感のあるデザイン
- リラクゼーションの雰囲気を保ちつつ、モダンでかっこいい
- 予約への導線を明確に
