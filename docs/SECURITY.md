# 🔒 セキュリティガイド

## 🛡️ セキュリティ対策

### **認証システム**
- Firebase Authentication を使用
- メール・パスワード認証
- 管理者権限のFirestore管理

### **アクセス制御**
- 管理者以外は管理画面にアクセス不可
- 認証状態のリアルタイム監視
- 適切なセッション管理

## 🔑 管理者アカウント管理

### **初期設定**
- **メール**: `admin@yoon2.com`
- **パスワード**: `admin123456`

### **本番環境での必須対応**
1. **パスワード変更**: 強力なパスワード（12文字以上）
2. **アカウント管理**: 不要なアカウントの削除
3. **アクセス監視**: 定期的なログ確認

## 🚨 セキュリティチェックリスト

### **✅ 必須項目**
- [ ] 管理者パスワードを変更済み
- [ ] 強力なパスワードを使用
- [ ] 不要な管理者アカウントを削除
- [ ] アクセスログを定期的に確認

### **🔍 定期確認項目**
- [ ] 管理者アカウントの一覧確認
- [ ] アクセスログの確認
- [ ] セキュリティ設定の見直し
- [ ] バックアップの確認

## 📞 セキュリティインシデント対応

### **疑わしいアクセスを発見した場合**
1. 即座にパスワードを変更
2. 不要なセッションを終了
3. アクセスログを確認
4. 必要に応じてアカウントを無効化

### **緊急時の対応**
1. Firebase Console でアカウントを無効化
2. Firestore で管理者権限を削除
3. システムの再設定

---

**重要**: セキュリティは継続的な取り組みです。定期的な見直しと更新を行ってください。
