## 🛠️ 技術スタックの拡張予定

### 現在の技術スタック
- **フロントエンド**: React 19.1.1, CSS3
- **バックエンド**: Firebase (Firestore, Hosting)
- **開発環境**: Create React App

### 追加予定の技術
- **状態管理**: Redux Toolkit または Zustand
- **UIライブラリ**: Material-UI または Chakra UI
- **テスト**: Jest, React Testing Library, Cypress
- **型安全性**: TypeScript
- **ビルドツール**: Vite (Create React App からの移行検討)
- **コンテナ**: Docker, Docker Compose
- **CI/CD**: GitHub Actions
- **監視**: Sentry, Google Analytics
