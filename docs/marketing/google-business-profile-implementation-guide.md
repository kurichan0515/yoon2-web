## 🔐 アカウント作成・ログイン

### ステップ1: Googleアカウントの確認

1. **Googleアカウントを持っている場合**
   - https://accounts.google.com/ にアクセス
   - ログインしてアカウントを確認

2. **Googleアカウントを持っていない場合**
   - 「アカウントを作成」をクリック
   - 必要事項を入力してアカウントを作成

### ステップ2: Googleビジネスプロフィールにアクセス

1. **パソコンから**
   - https://www.google.com/business/ にアクセス
   - 「今すぐ始める」または「管理」をクリック

2. **スマートフォンから**
   - Googleマップアプリを開く
   - メニュー → 「ビジネスを追加」または「ビジネスを管理」

### ステップ3: ビジネスプロフィールの作成または確認

1. **既にプロフィールが存在する場合**
   - 「ビジネスを管理」を選択
   - 既存のプロフィールを選択

2. **新規作成の場合**
   - 「ビジネスを追加」を選択
   - 「個別の店舗やサービス提供場所」を選択

## 📝 基本情報の登録

### ステップ1: 店舗名の登録

1. 「ビジネス名」欄に以下を入力：
   ```
   yoon²ゆんゆん
   ```
2. **重要**: 店舗名は統一してください（他のサイトでも同じ表記）

### ステップ2: カテゴリの設定

1. 「カテゴリ」をクリック
2. 主カテゴリを選択：
   - 「エステティックサロン」を検索して選択
3. サブカテゴリを追加（オプション）：
   - 「リラクゼーションサロン」を追加

### ステップ3: 住所の登録

1. 「住所」欄に以下を入力：
   ```
   愛媛県松山市清水町３丁目２０１－２ ８Ｄｒｏｐｓ １０２号室
   ```
2. 地図上で位置を確認・調整
3. **重要**: 住所は正確に入力してください

### ステップ4: 電話番号の登録

1. 「電話番号」欄に以下を入力：
   ```
   080-8478-1163
   ```
2. **重要**: ハイフンありで統一してください

### ステップ5: 営業時間の設定

1. 「営業時間」をクリック
2. 各曜日の営業時間を設定：
   ```
   月曜日: 10:00 - 20:00
   火曜日: 10:00 - 20:00
   水曜日: 10:00 - 20:00
   木曜日: 10:00 - 20:00
   金曜日: 10:00 - 20:00
   土曜日: 10:00 - 20:00
   日曜日: 10:00 - 20:00
   ```
3. 「営業時間の注記」に以下を追加：
   ```
   営業時間外でも対応できることがあります。お気軽にお問い合わせください。
   ```

### ステップ6: ウェブサイトURLの設定

1. 「ウェブサイト」欄に予約システムのURLを入力
2. 例: `https://yoon2.com` または予約ページのURL

### ステップ7: 所有権の確認

1. 所有権確認方法を選択：
   - **電話**: 電話番号で確認（推奨）
   - **メール**: メールアドレスで確認
   - **郵便**: 郵便物で確認
2. 確認コードを入力
3. 所有権が確認されるまで数分〜数日かかる場合があります

## ❓ FAQ設定

### ステップ1: FAQの追加

1. 「よくある質問」セクションを開く
2. 「質問を追加」をクリック
3. 以下の質問と回答を追加：

#### Q1: 初めて利用する場合の流れは？
**A**: まずはオンライン予約または公式LINEからご予約ください。初回はカウンセリングを行い、お客様のご希望やお悩みをお聞きします。その後、最適なメニューをご提案いたします。

#### Q2: 予約はどのようにすればいいですか？
**A**: オンライン予約システムからご予約いただけます。また、公式LINEからもご予約・お問い合わせが可能です。営業時間内であれば、お電話（080-8478-1163）でも承っております。

#### Q3: 営業時間外でも予約できますか？
**A**: はい、オンライン予約システムは24時間ご利用いただけます。営業時間外でも対応できることがありますので、お気軽にお問い合わせください。

#### Q4: 駐車場はありますか？
**A**: はい、駐車場を完備しております。北久米駅からも徒歩5分とアクセス良好です。

#### Q5: キャンセルポリシーは？
**A**: 施術直前のキャンセルは施術料の50%、無断キャンセルは施術料の100%をいただいております。ご理解の上、ご予約ください。

#### Q6: 男性も利用できますか？
**A**: はい、男性のお客様もご利用いただけます。男性向けのメニューもご用意しております。

#### Q7: 効果はどのくらいで実感できますか？
**A**: 個人差がありますが、多くのお客様が初回からリラックス効果を実感されています。継続的にご利用いただくことで、より効果を感じていただけます。

#### Q8: 予約の変更やキャンセルはできますか？
**A**: はい、可能です。オンライン予約システムから変更・キャンセルができます。お急ぎの場合は、公式LINEまたはお電話でご連絡ください。
