## 1. 市場分析 & SWOT分析

### 1.1 業界市場規模とトレンド

#### エステティックサロン市場全体
- **2024年度市場規模**: 約3,043億円（前年比-1.7%）
- **2025年度予測**: 約3,046億円（前年比+0.1%）
- **市場の課題**: コロナ禍以降の回復が不十分、脱毛サロン大手の倒産による顧客信頼の低下、過当競争と低価格競争の激化

#### 耳つぼ・イヤーエステ市場の特徴
- 専門特化型サロンとしてニッチ市場を形成
- 10～50代まで幅広い年齢層に支持
- 手軽さと効果の両立を求める顧客ニーズが高まっている
- オンライン予約利用率: 60%以上（業界平均）

#### 愛媛県松山市の市場環境
- 総合エステ店: 約20店舗
- メンズエステ店: 約19店舗
- アジアンエステ店: 約1店舗
- **競合店の価格帯**: 60分10,000円～11,000円（yoon²ゆんゆんは3,500円～12,500円と幅広い価格帯で差別化可能）

### 1.2 SWOT分析

#### 強み（Strengths）
1. **専門特化**: 耳つぼ・イヤーエステに特化した専門性
2. **価格競争力**: 初回3,500円～4,000円と手頃な価格設定
3. **オンライン予約システム**: オンライン予約が可能な独自システム
4. **柔軟な営業時間**: 10:00～20:00、営業時間外対応も可能
5. **立地**: 北久米駅から徒歩5分、駐車場完備
6. **多様なメニュー**: 耳つぼ、イヤーエステ、ドライヘッドスパの組み合わせ
7. **技術スタック**: モダンなWebアプリケーションで顧客体験を向上

#### 弱み（Weaknesses）
1. **認知度**: 新規店舗のため地域での認知度が低い可能性
2. **競合との差別化**: 松山市内に20店舗以上の競合が存在
3. **マーケティング予算**: 小規模サロンのため広告予算が限定的
4. **スタッフ数**: 少数精鋭体制のため、同時対応に制約
5. **口コミ数**: 開業間もないため、実績・口コミが少ない

#### 機会（Opportunities）
1. **デジタルマーケティング**: SEO対策が不十分な競合が多い
2. **Instagram集客**: 20～40代女性の利用率が高い（20代73.3%、30代63.3%）
3. **ローカルSEO**: Googleマイビジネス最適化で地域検索上位表示の可能性
4. **オンライン予約需要**: 60%以上のユーザーがネット予約を利用
5. **健康意識の高まり**: セルフケア・未病ケアへの関心が増加
6. **リピーター層**: 継続利用しやすい価格設定とサービス内容
7. **メンズ市場**: 松山市内にメンズエステ需要が存在

#### 脅威（Threats）
1. **市場縮小傾向**: エステサロン市場全体が縮小傾向
2. **価格競争**: 低価格競争の激化
3. **顧客信頼の低下**: 脱毛サロン大手の倒産による業界全体への不信感
4. **競合の参入**: 新規参入による競争激化の可能性
5. **経済状況**: 景気後退による顧客の支出削減

### 1.3 重要な発見と機会

1. **デジタルマーケティングの未開拓領域**: 多くの競合がSEO対策やSNS運用を十分に行っていない
2. **地域密着型マーケティングの有効性**: ローカルSEOとGoogleマイビジネス最適化で地域検索での上位表示が可能
3. **Instagramの集客力**: ターゲット層の利用率が高く、ビジュアル訴求が強い
4. **オンライン予約システムの優位性**: オンライン予約で機会損失を削減
5. **価格帯の柔軟性**: 幅広い価格帯で様々な顧客層に対応可能

## 6. リスク管理と代替案

### 6.1 リスク要因

1. **予算不足**: マーケティング予算が限定的
   - **対策**: 無料・低コスト施策を優先（SEO、SNS、口コミ）

2. **時間不足**: スタッフが少なく、マーケティングに時間を割けない
   - **対策**: 自動化ツールの活用、外部委託の検討

3. **競合の参入**: 新規参入による競争激化
   - **対策**: 差別化要素の強化、顧客ロイヤルティの構築

4. **市場の縮小**: エステサロン市場全体の縮小
   - **対策**: ニッチ市場での専門性強化、オンラインサービスの検討

### 6.2 代替案

1. **コンテンツ制作の外部委託**: 写真・動画制作を外部に委託
2. **SNS運用の自動化**: スケジューリングツールの活用
3. **アフィリエイトプログラム**: 紹介キャンペーンの実施
4. **地域パートナーシップ**: 他店舗とのコラボレーション
