# yoon2.com がGoogle検索に表示されない原因調査

調査日: 2026-02-11  
更新: Search Console「URL 検査」でインデックス済みを確認済み

## 重要: Search Console の結果（2026-02-11 確認）

- **URL は Google に登録されています**
- **ページはインデックスに登録済みです**
- **HTTPS で配信されています**

→ **インデックスされていないことが原因ではない**ため、**ランキング・検索クエリ・表示条件**が原因と考えられます。

---

## 調査結果サマリ（技術面）

| 項目 | 状態 | 備考 |
|------|------|------|
| robots.txt | ✅ 問題なし | `Allow: /`、Sitemap URL 記載、主要ページはブロックされていない |
| noindex | ✅ なし | メタに noindex は設定されていない |
| sitemap.xml | ✅ 存在 | 本番の `/sitemap.xml` はプロジェクトに定義あり |
| meta description / OGP | ✅ 設定済み | index.html に記載あり |
| canonical | ✅ 設定済み | `https://yoon2.com/` |

**結論（技術面）**: ブロックや設定ミスは見当たりません。**検索に「出てこない」主因はランキング・キーワード・競合です。**

---

## 想定される原因（インデックス済みである前提）

Search Console で**インデックス登録済み**が確認できたため、以下が主な要因です。

### 1. **ランキングが低い（順位が何ページも下）** 【最有力】

- ページはインデックスされていますが、**検索キーワードに対する順位が 2〜3 ページ目以降**にあり、普段の検索では見えない状態の可能性があります。
- 「松山 エステ」「耳つぼ 松山」などは競合が多く、新規・小規模サイトは上位に出にくいです。

### 2. **検索キーワードとページ内容の一致度**

- ユーザーが使っているキーワード（例: 「yoon2」「ゆんゆん」）と、ページのタイトル・本文・メタの一致が弱いと、表示されにくくなります。
- 店名・サービス名が本文に十分含まれているか確認するとよいです。

### 3. **サイトの権威性・新しさ**

- ドメインやサイトが新しく、被リンクや実績が少ないと、同じキーワードで競合に負けやすくなります。
- 時間の経過とコンテンツ・被リンクの蓄積で改善していく部分です。

### 4. **地域・端末・パーソナライズ**

- 検索結果は地域・言語・ログイン状態で変わります。**シークレットや別端末で `site:yoon2.com`** を試すと、実際にインデックスされているか確認できます。

### 5. **手動による対策（ペナルティ）**

- Search Console の「手動による対策」に通知がなければ、通常は該当しません。気になる場合は GSC 左メニューで確認してください。

## 推奨アクション（インデックス済み＝ランキング対策中心）

### すぐにできる確認

1. **`site:yoon2.com` で表示を確認**
   - Google で `site:yoon2.com` と検索し、トップページなどが 1 件でも出ていれば、**インデックスは問題なし**です。  
     「出てこない」と感じているキーワード（例: 店名・サービス名）で、何ページ目に出るかを確認すると原因の切り分けになります。

2. **Search Console「検索パフォーマンス」の確認**
   - 左メニュー「パフォーマンス」で、**どの検索キーワードで表示・クリックされているか**を確認する。
   - 「表示回数はあるがクリックされていない」「表示回数がほぼない」なら、キーワード選定やタイトル・説明文の改善が有効です。

3. **手動による対策の有無**
   - Search Console の「手動による対策」に通知がなければ、ペナルティは考えにくいです。

### ランキング・露出を上げる対策

4. **タイトル・説明文の見直し**
   - 店名「yoon²ゆんゆん」や「耳つぼ」「イヤーエステ」「松山」などを、`public/index.html` の `<title>` と `meta name="description"` に自然な形で含める（すでに入っていれば、検索で使いたいキーワードの優先度を微調整）。

5. **本文テキストの充実**
   - トップページに、**検索されそうな語（地域名・サービス名・店名）が文章として**多く含まれるようにする。SPA のため、重要な語はできるだけ初期表示されるブロックに書く。

6. **地域SEO・口コミ**
   - Google ビジネスプロフィール（旧 Google マイビジネス）の登録・情報の最新化、口コミの蓄積は、地域検索の順位に効きやすいです。

7. **被リンク・SNS**
   - 公式 LINE・Instagram などからのリンク、地域サイトやブログでの紹介があると、権威性が上がり検索結果での露出が増えやすくなります。

### 余裕があればの技術対策

8. **SSR/プリレンダリングの検討**
   - 現状は SPA のため、**Next.js への移行（SSR/SSG）** や **プリレンダリング** を入れると、検索エンジンに本文が伝わりやすくなり、長期的な SEO に有利です。

9. **表示速度の確認**
   - トップページの表示が遅いとランキングに悪影響することがあります。Lighthouse などで計測し、必要なら軽量化を検討します。
